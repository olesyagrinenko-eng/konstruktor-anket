# Конструктор анкет (тесты креативов)

Веб-сервис на **Flask**: пошаговый мастер для сборки структуры анкеты под типы материалов (ролик, макет, сценарий, концепция, упаковка), этап претест/посттест, набор типовых показателей и доп. блоков.

Текущая версия умеет:

- собирать анкету по шаблонам в интерфейсе;
- импортировать готовую анкету из **Word (.docx)**;
- показывать редактируемую промежуточную структуру;
- конвертировать её в **SSI JSON** для Sawtooth / `SSIGen`;
- валидировать итоговый JSON по схеме `docs/contractor-ssi-questionnaire.schema.json`;
- выгружать и **DOCX**, и **SSI JSON**.

## Запуск локально

```bash
cd konstruktor-anket
pip install -r requirements.txt
python app.py
```

Откройте в браузере: `http://127.0.0.1:5050` (порт можно задать переменной `PORT`).

## API

| Метод | Путь | Описание |
|--------|------|----------|
| GET | `/api/catalog` | Справочник групп показателей и доп. опций |
| POST | `/api/suggest-groups` | Рекомендуемые `group_ids` по черновику payload |
| POST | `/api/build` | JSON спецификация анкеты (`meta` + `blocks`) |
| POST | `/api/import/docx` | Импорт `.docx` в редактируемую спецификацию + SSI JSON |
| POST | `/api/convert/ssi-json` | Конвертация внутренней спецификации в SSI JSON |
| POST | `/api/validate/ssi-json` | Проверка SSI JSON по JSON Schema |
| POST | `/api/export/docx` | Тело — та же спецификация (после правок в UI), ответ — файл `.docx` |
| POST | `/api/export/ssi-json` | Скачивание итогового SSI JSON |

## Структура проекта

- `app.py` — маршруты Flask  
- `catalog.py` — методология: группы показателей и шаблоны вопросов  
- `builder.py` — сборка спецификации с ротацией по числу стимулов  
- `docx_export.py` — выгрузка в Word  
- `word_import.py` — импорт из Word `.docx`  
- `ssi.py` — маппинг во внешний SSI JSON и валидация по схеме  
- `templates/index.html`, `static/` — интерфейс мастера  
- `docs/` — спецификация интеграции Sawtooth / SSI, схема и примеры  

Блок **скрининга** (`screening_base`) всегда включается в анкету на сервере, даже если снять другие группы.
